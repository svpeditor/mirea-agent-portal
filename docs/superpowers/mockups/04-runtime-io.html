Runtime-протокол — как агент общается с порталом

Контракт между процессом-агентом внутри Docker и воркером портала снаружи. Простой, текстовый, отлаживаемый tail -f.

🟦 Портал (worker)

  • Готовит $INPUT_DIR с загруженными файлами
  • Готовит $OUTPUT_DIR (пустой)
  • Запускает контейнер с инжектом env: OPENROUTER_API_KEY, OPENROUTER_BASE_URL
  • Кладёт параметры формы в $PARAMS_FILE (JSON)
  • Слушает stdout построчно — каждая строка JSON-событие
  • Транслирует события в WebSocket для UI и пишет в БД
  • По завершении — индексирует артефакты из $OUTPUT_DIR
→params,
files
←events,
artifacts

🟧 Агент (внутри Docker)

  • Читает параметры из $PARAMS_FILE
  • Читает входные файлы из $INPUT_DIR/<input_id>/...
  • Дёргает LLM через OpenRouter обычным OpenAI-клиентом
  • Пишет события прогресса в stdout (NDJSON)
  • Произвольный лог в stderr (туда же print, traceback)
  • Кладёт артефакты в $OUTPUT_DIR с именем из манифеста
  • Финальное событие result или failed

События в stdout — NDJSON-стрим

started
Агент стартанул. Портал ставит job в статус «выполняется».
progress
Числовой прогресс 0.0–1.0 плюс человеко-читаемая подпись. UI показывает прогресс-бар.
log
Сообщение пользователю в общий лог. Уровень info / warn / error / debug.
item_done
Завершение одного элемента в серии (статья / работа). Появляется в UI отдельной строкой со «✓».
error
Нефатальная ошибка — отдельный элемент не удался, агент продолжает. UI помечает элемент красным.
result
Успешное завершение. Список артефактов с id и относительными путями. UI открывает страницу скачивания.
failed
Критическое падение. Job помечается failed. Артефакты, если успели создаться, доступны для дебага.

Студент видит SDK, не сырой JSON

agent.py — пример с SDK портала
from portal_sdk import Agent from openai import OpenAI agent = Agent() # читает $PARAMS_FILE, $INPUT_DIR, $OUTPUT_DIR params = agent.params # dict с полями формы section = params["section"] works_dir = agent.input_dir("works_folder") # Path к папке с работами llm = OpenAI( api_key=agent.env["OPENROUTER_API_KEY"], base_url=agent.env["OPENROUTER_BASE_URL"], ) folders = sorted(works_dir.iterdir()) for i, folder in enumerate(folders): agent.progress(i / len(folders), f"Обработка {folder.name}") try: result = analyze_work(llm, folder, section) agent.item_done(folder.name, summary=result["verdict"]) except Exception as e: agent.error(folder.name, str(e)) agent.write_docx("экспертное_заключение.docx", ...) agent.result(artifacts=["report"]) # id из манифеста
